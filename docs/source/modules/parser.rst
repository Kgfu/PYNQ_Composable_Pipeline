pynq_composable.parser Module
=============================

The ``pynq_composable.parser`` module parses the metadata in the hwh files for
the full bitstreams and partial bitstreams and provides the ``c_dict`` and
``dfx_dict`` dictionaries that are paramount to interact with the composable
overlays.
